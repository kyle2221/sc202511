# **App Name**: Sourced Code

## Core Features:

- Vibe Input: Accept natural language descriptions of desired aesthetic vibes to guide code generation.
- Code Snippet Generation: Generate CSS snippets and design tokens from user-described vibes, employing a tool that leverages an LLM for contextual understanding.
- Real-time Preview: Provide a real-time preview of the generated styles as the user adjusts parameters or inputs new descriptions.
- Editable Outputs: Enable users to directly edit the generated CSS and design tokens for fine-tuning and customization.
- Export/Share Functionality: Allow users to export the generated code or easily share their sourced code creations.

## Style Guidelines:

- Color scheme: Dark background (#121212) to provide a modern coding environment, enhanced by electric blue accents, mirroring the v0.dev aesthetic.
- Primary color: Electric Blue (#7DF9FF), a vibrant hue that signifies innovation and digital aesthetics.
- Accent color: Complementary lighter blue (#B0E2FF) to highlight interactive elements and call-to-action buttons, offering a subtle but effective contrast.
- Body and headline font: 'Space Grotesk', a sans-serif font known for its modern, technical aesthetic.
- Code font: 'Source Code Pro', specifically chosen for its excellent legibility and suitability for displaying code.
- Use clean, minimalist icons in electric blue, with subtle animations on hover.
- Maintain a clean, grid-based layout that mirrors professional code editing environments.
